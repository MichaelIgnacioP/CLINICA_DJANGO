PARA INCIAR EL PROYECTO SEGUIR LAS SIGUIENTES INSTRUCCIONES:


1 .- eliminar carpeta venv actualmente creada
2 .- instalar entorno virtual con comando pip install virtualenv
3 .- crear entorno virtual python -m venv venv
4 .- activar entorno virtual ./venv/Scripts/activate
5 .- pip install -r requirement.txt
6 .- python manage.py makemigrations
7 .- python manage.py migrate
8 .- python manage.py runserver
